Ethical Hacking Roadmap

Your 2.5-year guide to becoming a job-ready ethical hacker

Year 1: Foundations and Basics

Months 1-4: Networking and Linux Fundamentals

  • Networking: TCP/IP, DNS, ARP, VPNs, Subnetting
  • Linux: Shell scripting, user permissions, file systems
  • Cisco Networking Academy
  • Linux Command Line

Months 5-8: Cybersecurity Basics and Tools

  • Threats and vulnerabilities: Malware, social engineering
  • Tools: Nmap, Wireshark, Metasploit
  • Hack The Box

Months 9-12: Start Penetration Testing

  • Phases: Reconnaissance, scanning, exploitation
  • Common attacks: SQL injection, XSS, CSRF
  • TryHackMe Pathways

Year 2: Intermediate and Specialization

Months 13-16: Web Application Security

  • OWASP Top 10: Input validation, session management
  • PortSwigger Web Security Academy

Months 17-20: Advanced Topics

  • Reverse shells, buffer overflows, privilege escalation
  • Exploit-DB

Months 21-24: Bug Bounties and Advanced Labs

  • Join platforms like HackerOne and Bugcrowd
  • Bug Bounty Hunter Methodology

Year 3: Job Preparation and Final Touches

Months 25-28: Certifications

  • Certified Ethical Hacker (CEH)
  • Offensive Security Certified Professional (OSCP)
  • CEH Guide
  • OSCP Prep

Months 29-30: Build a Portfolio

  • Showcase labs, bug bounties, and certifications
  • Write blogs or create video tutorials

Months 31-36: Apply and Network

  • Attend cybersecurity meetups and conferences
  • Apply for entry-level roles like SOC Analyst or Junior Pentester
